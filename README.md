# Careerly-AI

**Your Intelligent AI-Powered Career Roadmap Builder**

CareerForge AI helps students, professionals, and career switchers create a clear, personalized, and actionable roadmap to reach their dream job role — whether it's "Senior Software Engineer at Google", "Data Scientist", "Product Manager", or any other designation.

### ✨ Key Features
- **AI-Powered Roadmap Generation**: Enter your target job and current skills — get a complete structured roadmap with phases, milestones, required skills, and estimated timeline.
- **Interactive Progress Tracker**: Mark tasks complete, track overall progress, maintain streaks, and log notes.
- **Daily Personalized Suggestions**: Get smart daily course, video, or action recommendations based on your current progress.
- **Market Intelligence**: Real-time internship and job opportunity suggestions with match scores based on current market trends.
- **Beautiful Modern Dark UI**: Minimalistic, premium design with smooth Framer Motion animations and Lucide icons.
- **Secure Authentication** powered by Supabase.

### 🛠️ Tech Stack
- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS, shadcn/ui
- **Animations**: Framer Motion
- **Database & Auth**: Supabase
- **AI Engine**: Anthropic Claude (Claude 3.5 Sonnet / Opus)
- **Charts**: Recharts
- **Icons**: Lucide React

### 🎨 Design
Modern dark minimalistic theme with clean typography, subtle glassmorphism, smooth micro-interactions, and premium feel.
